# Desktop Environemnt Setting

## Batter Alert 

 Create a battery alter by consulting [1]. 

 * Setup icon at an appropriate location, e.g. /usr/share/icons/...

 * Change sound alter command to `spd-say -w '<the message goes here>'`.
